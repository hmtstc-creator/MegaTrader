`Merhaba Ahmet` metnini bir dosyaya yazmak için aşağıdaki komutu kullanabilirsiniz.

**1. Yeni bir dosya oluşturmak veya mevcut dosyanın içeriğini silip yenisiyle değiştirmek için:**

```bash
echo "Merhaba Ahmet" > test_dosyasi.txt
```

*   **`echo "Merhaba Ahmet"`**: Bu kısım, tırnak içindeki metni ekrana (standart çıktıya) yazar.
*   **`>`**: Bu işaret, `echo` komutunun çıktısını standart çıktı yerine belirtilen dosyaya yönlendirir. Eğer `test_dosyasi.txt` adında bir dosya yoksa oluşturur, varsa içeriğini tamamen siler ve "Merhaba Ahmet" metniyle değiştirir.

**2. Mevcut bir dosyanın sonuna metni eklemek için (içeriğini silmeden):**

```bash
echo "Merhaba Ahmet" >> test_dosyasi.txt
```

*   **`>>`**: Bu işaret, `echo` komutunun çıktısını belirtilen dosyanın sonuna ekler. Dosya yoksa oluşturur.

---

**Örnek kullanım ve kontrol:**

1.  **Komutu çalıştırın:**
    ```bash
    echo "Merhaba Ahmet" > test_dosyasi.txt
    ```

2.  **Dosyanın içeriğini kontrol edin:**
    ```bash
    cat test_dosyasi.txt
    ```
    Bu komutun çıktısı şöyle olacaktır:
    ```
    Merhaba Ahmet
    ```
